# Codigos Linux

---

**LS**- LISTAR CONTEUDOS EM UMA PASTA

**CD**- ENTRAR DENTRO DAS PASTAS

**MKDIR**- CRIAR PASTAS

**CLEAR**- LIMPAR O CMD

**MAN**- MANUAL (man ls)

PWD- MOSTRA QUE LUGAR (PASTA) ESTAMOS

WHOAMI- NOME DO USUARIO

- whoaim >> ARQUIVO.txt

o  >> “entra/joga” no arquivo

TOUCH- NORMALMENTE USADO PRA CRIAR PASTAS VAZIAS

- EX: TOUCH “OIUTUDOBEM?” VAI CMC UM ARQUIVO NOVO COM ESTE NOME

NANO- É UM EDITOR DE TEXTOS

- CTRL + X = SALVAR

CAT- CONTEUDO DE ARQUIVOS/ É USADO PARA VR OQ TEM ENRO DE UM ARQUIVO 

CHMOD- ELE MOSTRA QUEM POE FAZER OQ COM OS ARQUIOS, TIPO LER, ESCREVER, EXECUTAR

- r (read)- pode ler o arquivo
- w (write)- pode escrever
- x (execute)- pode executar/rodar (so faz sentido pra scripts/programas)
- o “-” significa q nao tem permissão

TAIL- MOSTRA AS ULTIMAS 10 LINHAS DO COD

- o (-f) fica atualizando automaticamente

GREP- É USADO PRA PROCURAR UM TEXTO DENTRO DO ARQUIVO 

- grep "ERRO" arquivo.log

ELE MOSTRA TODOS OS “ERRO” DENTRO DO COD

FFUF- AJUDA A ENCONTRAR PAGINAS ESCONDIDAS EM SITES 

sudo -l: ver como root
